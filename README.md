## Внедрение методов снижения дисперсии в А/В-тестировании.

Целью работы было провести синтетический А/В-тест E2E CR с использованием CUPED, стратификации и сравнить с базовым подходом для z-test для долей. 
Датасет с платформы Kaggle состоял из 5 таблиц, соотв. месяцу в период с 10.2019 по 02.2020. 

Для симуляции эксперимента предварительно была выполнена очистка данных, EDA, выбор периода и предпериода для симуляции эксперимента, анализ стабильности конверсии по продуктовой воронке и поиск ковариаты.
Результаты показали, что CUPED и стратификация смогли обнаружить синтетический эффект в отличии от z-test для долей. CUPED снизил дисперсию целевой метрики на 5.29%.
